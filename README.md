# rpispeedlog
Raspberry Pi Internet Speed Test Logger was just an experiment to learn the basics of Matplotlib and other Python modules. Purpose was to obtain results from speedtest-cli at scheduled cron interval, store results using SQLITE3, plot graph images using Python Matplotlib and display graph image along with selectable database logs on webpage using Apache and PHP.
